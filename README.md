# Accounting Automation App
## Overview
This project is designed to automate daily Accounts Payable (AP) QA reporting processes. 
## Problem 
Manual data extraction and reporting is time-consuming and prone to human errors. 
## Solution
This automation script streamlines data handling and generates structured reports efficiently. 
## Key Features
- Automates repetitive accounting tasks
- Improves data accuracy
- Reduces manual effort
- Speeds up reporting process
## Tools Used
- Python
- Excel
- Selenium
